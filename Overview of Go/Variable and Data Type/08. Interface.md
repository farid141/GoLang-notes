# Interface

Interface di Golang adalah kumpulan definisi method tanpa implementasi.

```go
type HasName interface{
 GetName() string
}
```

## Implementasi Interface

Tipe data apapun yang `mengimplementasikan semua fungsi abstrak` interface akan dianggap memenuhi interface tersebut. Bisa berupa `struct, pointer ke struct dan tipe lain`.

Interface memenuhi nil, artinya kita bisa mengembalikan nil kalau tidak ingin buat interface.

## Compare interface dengan struct

sebuah struct dapat dianggap implementasi interface jika memiliki kontrak yang sesuai dengan interface.

```go
type Speaker interface {
    Speak() string
}

type Person struct {
    Name string
}

// Person mengimplementasi interfacfe Speaker
func (p Person) Speak() string {
    return "Hi"
}

func main() {
    var i Speaker = Person{Name: "Alice"}
    p := Person{Name: "Alice"}

    fmt.Println(i == p) // âœ… true: dynamic type and value match

    // compare lebih aman
    _, ok := someValue.(Speaker)
    if ok {
        fmt.Println("It implements Speaker")
    }
}
```
