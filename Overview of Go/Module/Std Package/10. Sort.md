# Sort

Berguna untuk pengurutan data

## Kontrak untuk Sort

```go
type Interface interface{
    Len() int
    Less(i, j int) bool // fungsi pengurutan menentukan apakah i harus sebelum j 
    Swap(i, j int) // menukar posisi
}

type User struct{
    name string
    age int
}

type UserSlice []User

func (u UserSlice) Len() int{
    return len(u)
}
func (u UserSlice) Less(i, j int) bool{
    return u[i].Age > u[j].Age
}
func (u UserSlice) Swap(i, j int){
    u[i], u[j] = u[j], u[i]
}

users := []User{
    {"Eko", 24},
    {"Budi", 23},
    {"Utomo", 22},
}

sort.Sort(UserSlice(users))
```
