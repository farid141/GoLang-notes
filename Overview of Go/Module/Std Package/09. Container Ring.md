# Container Ring

Merupakan struktur data circular-list dimana head=tail (`Next()` tidak akan bernilai nil). Konsepnya sama seperti container list.

```go
data := list.New()
data.PushBack("farid")
data.PushBack("nubaili")
data.PushBack("123")

for e:=data.Front(); e < len(data); e=e.Next(){
    // next mengembalikan pointer struct selanjutnya, sehingga kita bisa akses member dengan .value
    fmt.Println(e.Value)
}
```
