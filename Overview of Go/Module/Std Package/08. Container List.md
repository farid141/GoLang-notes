# Container List

Digunakan untuk membuat struktur data double-linked-list. Digunakan untuk sebuah antrean

Data dalam sebuah linked-list berupa any (boleh tidak seragam).  

```go
data := list.New()
data.PushBack("farid")
data.PushBack("nubaili")
data.PushBack("123")

for e:=data.Front(); e != nil; e=e.Next(){
    // next mengembalikan pointer struct selanjutnya, sehingga kita bisa akses member dengan .value
    fmt.Println(e.Value)
}
```

Kita bisa mengakses (write & read ke .Value)
