digunakan pada fungsi go routine untuk mengirim data antar go routine

sebuah goroutine yang mengeluarkan/mengambil data channel dalam fungsinya akan menunggu sampai data channel diterima/tersedia (seperti await)

karakteristik channel:
- tidak bisa mengirim data ke channel sampai channel kosong
- channel hanya bisa menerima satu jenis data (jika masih pakai interface, maka penerima harus mampu mengkonversi tipe data tersebut (tidak disarankan))
- bisa diambil atau dikirim dari lebih satu go routine
- channel harus diclose agar tidak memory leak:
> defer close(channel_var)

# membuat channel, mirip dengan map:
var channel: chan = make(chan tipe_data)

# mengirim data
channel <- "Eko"

# menerima data
data := <- channel

# error pada channel:
- tidak ada penerima data: panic
- tidak ada pengirim data: deadlock

# channel sebagai parameter
func func_name(channel chan <type>)

# memberitahu param channel sebagai pengirim/penerima
(channel chan<- <type>)
